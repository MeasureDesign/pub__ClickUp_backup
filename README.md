# ClickUp Backup Toolkit

This repository contains tools for working with ClickUp data outside the platform. It brings together three separate parts of the workflow: backup, restore, and viewer.

## Repository structure

The root of the repository contains three folders:

- `backup` – code for creating ClickUp backups
- `restore` – code for restoring backup data into ClickUp
- `viewer` – code for viewing backup contents outside ClickUp

Each of these folders includes its own dedicated documentation with usage details, requirements, and setup instructions.

## Purpose

The goal of this repository is to provide a more complete way to preserve and work with ClickUp data by:

- creating portable backups
- restoring backed-up data when needed
- viewing exported content independently of ClickUp
