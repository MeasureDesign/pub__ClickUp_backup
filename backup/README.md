# ClickUp Backup Script

This repository contains a Python-based backup script for exporting ClickUp workspace data into a structured backup format. The script is designed to help preserve important ClickUp content such as spaces, lists, tasks, comments, and related assets for archival and recovery purposes.

The repository is divided into two main subfolders:

- `google_cloud_backup/` – contains the version prepared for deployment and execution in Google Cloud
- `local_backup/` – contains the version intended for local use on Linux systems

Each of these subfolders includes its own dedicated `README.md` file with environment-specific setup, configuration, and usage instructions.
